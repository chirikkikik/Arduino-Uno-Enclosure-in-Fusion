# Arduino-Uno-Enclosure-in-Fusion

Designed a portable enclosure for an Arduino Uno to protect the board and provide easy access to USB and power ports.

## Overview
The enclosure consists of two parts (top and bottom), connected using integrated standoffs and pegs.

## Features
- 3D-printable design
- ABS material
- Wall thickness: 4 mm
- Clearance: 0.5 mm
